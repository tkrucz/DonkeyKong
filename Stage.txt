Game Stage File Format
Signs meaning:
P,x,y,w,z - platform, x - column spawn point cordinate, y - row spawn point cordinate, w - platform width, z - platform length
L,x,y,z - ladder, x - column spawn point cordinate, y - row spawn point cordinate, z - ladder height
B,x,y - barrels, x - column spawn point cordinate, y - row spawn point cordinate
T,x,y - trophies, x - column spawn point cordinate, y - row spawn point cordinate
* - end of reading sign
